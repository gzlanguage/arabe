1. هدف التطبيق

2. الوصول واستخدام الخدمات

يخضع استخدام تطبيق G-Z للقواعد التالية:

  • مجانًا: جميع الميزات والبيانات في التطبيق مجانية ومتاحة للجميع. لا ينبغي أن يتم monetized أي معلومات أو بيعها، بشكل مباشر أو غير مباشر.
  • للاستخدام الشخصي أو التعليمي: التطبيق مصمم للاستخدام الشخصي أو التعليمي أو البحث غير التجاري. أي محاولة لاستخدام بياناتنا أو خدماتنا لأغراض تجارية أو ربحية محظورة بشدة.

3. قيود الاستخدام

عند استخدام GZ، فإنك توافق على:

  1. عدم استخدام بياناتنا أو خدماتنا لأغراض احتيالية أو غير قانونية أو غير أخلاقية.
  2. عدم إعادة إنتاج أو توزيع أو بيع أو استغلال البيانات التي تم إنشاؤها بواسطة تطبيقنا لأغراض تجارية أو ربحية.
  3. عدم الوصول إلى أو التلاعب أو محاولة التدخل في أنظمة أو خوادم G-Z بطريقة غير مصرح بها.
  4. احترام هدف الشفافية والوصول المجاني للتطبيق بعدم المطالبة بأي حقوق ملكية على البيانات الناتجة.

4. حقوق الملكية الفكرية

تظل جميع المحتويات والأكواد والتصاميم أو الوظائف الخاصة بتطبيق G-Z ملكية حصرية لمبدعيها. لا تكتسب أي حقوق ملكية من خلال استخدام التطبيق. يُسمح لك بالاستشارة، التحميل، واستخدام الإحصائيات فقط وفقًا للقواعد المذكورة أعلاه.

5. حماية البيانات والخصوصية

تلتزم GZ بحماية المعلومات التي تقدمها وفقًا لسياسة الخصوصية الخاصة بنا.

  • يتم تجميع جميع البيانات على الفور وإخفاء الهوية.
  • نحن لا نجمع أو نخزن معلومات يمكن أن تحدد هوية المستخدمين الفرديين.
  • أي محاولة لإعادة تعريف الهوية أو استخدام البيانات لتحديد فرد محظورة بشدة ويمكن أن تؤدي إلى عقوبات قانونية.

6. مسؤوليات المستخدم
